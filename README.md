To checkout my splash screen edit:  
1) Backup

```
mkdir ~/SplashBackup
sudo cp -R /usr/share/plasma/look-and-feel/com.github.varlesh.adapta ~/SplashBackup
```
2) Copy my edited splash screen:

```
sudo cp -R usr/ /
```
